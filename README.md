# 光子四象限 (Photon Quadrants)

本地优先 · 极简设计 · 无需联网
一个基于四象限法则的网页端任务管理工具，帮助你把时间花在重要的事情上。
✨ 核心功能

* 经典四象限布局：将任务分为「紧急重要」、「紧急不重要」、「重要不紧急」、「不紧急不重要」，直观管理优先级。
* 拖拽排序：支持在同一象限内拖拽调整顺序，或跨象限拖拽移动任务。
* 极简交互：
   * 单击任务：标记完成/查看详情。
   * 长按任务（移动端）：标记/取消「今日专注」。
   * 编辑模式：开启后可单击任务直接编辑内容或备注。
* 专注模式：一键过滤，仅显示被标记为「今日专注」的任务。
* 数据掌控：
   * 本地存储：所有数据保存在浏览器本地，隐私安全。
   * 导入/导出：支持 Excel (.xlsx) 和 CSV 格式，方便备份与迁移。
   * 手动备份：一键备份当前状态，支持回滚恢复。

📖 使用指南

基础操作

* 添加任务：点击任意象限底部的 
"+ 添加" 按钮。
* 完成任务：单击任务项前的复选框或直接点击任务文字。
* 移动任务：按住任务并拖动到目标象限。
* 查看详情：单击任务文字区域，弹出详情浮层。

模式切换

* 🎯 专注今日：顶部工具栏开启，只显示带有 
"🎯" 标记的任务。
* ✏️ 编辑模式：开启后，单击任务直接进入编辑界面，适合修改长文本。
* ☰ 批量管理：开启多选模式，可对多个任务进行移动或删除。

添加到手机桌面 (PWA)

* iPhone (Safari)：打开页面 → 点击底部「分享」→「添加到主屏幕」。
* Android (Edge/Chrome)：打开页面 → 点击菜单 →「添加至手机」。

⚙️ 设置与选项

点击右上角 
"⚙️" 进入设置面板：

* 紧凑模式：增大信息密度，在一屏内显示更多任务。
* 显示已完成：切换是否隐藏已完成的任务。
* 表格视图：以表格形式查看所有数据。
* 数据导出/导入：位于设置面板底部。

💾 数据存储说明

* 存储位置：数据完全存储在浏览器的 
"localStorage" 中。
* 隐私安全：不会收集任何个人信息，无服务器后端，任务数据永不外传。
* 风险提示：清除浏览器缓存或将工具从主屏幕移除可能会导致数据丢失，请务必定期使用「手动备份」或「导出文件」功能。
